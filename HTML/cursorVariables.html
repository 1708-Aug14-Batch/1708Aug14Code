A cursor variable is, as you might guess from its name, a variable that points to a cursor or a result set.


Unlike with an explicit cursor, you can pass a cursor variable as an argument to a procedure or a function.


Cursor variables can be used with either embedded (static) or dynamic SQL. Below we see a function which returns a cursor variable that fetches either employee or department names, depending on the argument passed to the function.


CREATE OR REPLACE FUNCTION names_for (name_type_in IN VARCHAR2)

RETURN SYS_REFCURSOR

IS

    l_return SYS_REFCURSOR;

BEGIN

    CASE name_type_in

    WHEN 'EMP' THEN

        OPEN l_return FOR

        SELECT last_name

        FROM employees

        ORDER BY employee_id;

    WHEN 'DEPT' THEN

        OPEN l_return FOR

        SELECT department_name

        FROM departments

        ORDER BY department_id;

    END CASE;


    RETURN l_return;

END names_for;
